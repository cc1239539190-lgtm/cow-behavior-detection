## 介绍

基于YOLO目标检测系列的奶牛行为检测系统，本牛只智能检测系统适用于各类规模化奶牛、肉牛养殖牧场、集约化畜禽养殖基地、乡村畜牧养殖园区以及畜牧养殖产业园区等各类畜牧养殖经营场所，可满足养殖管理人员、畜牧技术工作人员、园区运维人员及基层畜牧监管人员日常工作使用需求，既能够对现场拍摄的牛只实景图片、养殖场所留存的养殖监控视频文件进行离线智能识别分析，也可接入养殖园区本地摄像头画面、园区实时监控录像流开展全天候在线实时智能监测，可实现养殖场内牛只数量自动统计、牛只站立、行走、卧躺等行为姿态智能识别研判，同时还可应用于畜牧养殖智能化管理配套运维、畜牧养殖行业数字化调研统计、农业智能视觉技术教学实训、养殖智能监测项目功能演示以及畜牧相关智能识别技术实操测试等多种场景，软件适配普通办公计算机、园区监控管理主机等主流设备运行环境，可广泛服务于畜牧养殖智能化升级、养殖场景无人化巡检、牲畜健康状态初步筛查等相关业务工作。
